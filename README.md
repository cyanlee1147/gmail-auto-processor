# Gmail 自動信件處理工具

## 功能說明
根據預設規則，自動處理收件匣中的信件，包含：
- 將指定信件**轉寄**給對應的負責人
- 自動套用 **Gmail 標籤**分類
- 將不需要的信件移至**垃圾桶**
- 處理完畢後自動**封存**

## 使用方式
1. 在 Google Apps Script 貼上程式碼
2. 修改 `CONFIG` 區塊中的 email 地址為實際收件人
3. 確認 Gmail 中已建立對應的標籤（安達、安聯、新光⋯⋯等）
4. 設定觸發器定時執行 `processEmails()`

## 注意事項
- 程式碼中的 email 地址請填入實際收件人，範例中使用佔位符
- 建議設定每天或每小時自動觸發

# Gmail 信件匯出工具

## 功能說明
從 Gmail 指定標籤中，將所有信件內容（日期、寄件人、主旨、內文）批次匯出成 Excel 檔案。

## 使用方式
1. 在 Google Apps Script 貼上程式碼
2. 修改 `LABEL_NAME` 為你的 Gmail 標籤名稱
3. 執行 `startExport()` 開始匯出
4. 匯出完成後，Excel 檔案會自動儲存到 Google Drive

## 注意事項
- 信件數量較多時，程式會自動分批執行（每次最多跑 5 分鐘）
- 每 10 分鐘自動續跑，直到全部完成
